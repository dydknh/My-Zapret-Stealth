# My-Zapret-Stealth
Обход блокировок для Ростелекома и других провайдеров
# Инструкция по запуску
1. Положите папку zapret в корень локального диска C: (путь должен быть `C:\zapret`).
2. Запустите файл Zapret_VPN.exe от имени администратора.
3. Нажмите кнопку ЗАПУСТИТЬ.
4. Закройте окно приложения — оно автоматически свернется в трей (рядом с часами) и продолжит работу.
